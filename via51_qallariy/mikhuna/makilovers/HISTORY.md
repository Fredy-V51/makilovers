# 📓 Bitácora de Ingeniería: Transición a la Nube

Este documento registra la evolución técnica del proyecto Makilovers y las decisiones arquitectónicas tomadas por **Via51**.

## Fase 1: Génesis Local
- **Desarrollo Inicial:** Creación de `index.html` con diseño moderno (Flexbox/Gradientes).
- **Optimización de Recursos:** Uso de **Git GUI** en entorno local para minimizar el consumo de RAM, priorizando procesos de *Stage*, *Commit* y *Push* hacia GitHub.

## Fase 2: El Salto Antigravity (Cloud)
- **Migración a Codespaces:** Se eliminó la fricción del hardware físico activando el entorno virtual de GitHub.
- **Ventaja Operativa:** Desarrollo continuo sin interrupciones por memoria o procesador, permitiendo pruebas en tiempo real.

## Fase 3: Despliegue Profesional
- **Configuración Firebase:** Vinculación exitosa del proyecto con Google Cloud.
- **Superación de Errores:** Resolución de conflictos de API (404) mediante la activación manual de servicios de Hosting y vinculación de Site IDs.
- **Estado Actual:** Primera versión estable desplegada en [via51-makilovers.web.app](https://via51-makilovers.web.app).
